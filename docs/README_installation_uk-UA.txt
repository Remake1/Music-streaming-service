    1.0) ВІДКРИЙТЕ ПАПКУ ПРОЕКТА У ТЕРМІНАЛІ/CMD
    1.1) ЯКЩО ВИ ЗАПУСКАЙТЕ САЙТ ПЕРШИЙ РАЗ ТА ПАКЕТИ НЕ ВСТАНОВЛЕННІ:
